A countdown timer to measure the time until an event happens.

## Features

- Measuring the time until an event happens.
- Counting time in Reverse.

## Technologies

- JavaScript
- React.js
- Vercel

## Getting Started

To get started with this project, you can clone the repository and install the
dependencies:

```bash
git clone
cd to the local address of the directory which you saved the project files.
npm install
```

Once the dependencies are installed, you can run the development server:

```bash
npm run dev
```

The development server will start at `http://localhost:3000`.

## Deployment

This appl is deployed to Vercel and available to check here : https://vercel.com/maryamyasaeis-projects/countdown-timer-app
